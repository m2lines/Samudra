# Quickstart configs

Tiny Samudra training that fits free-tier Google Colab (T4, 16 GB VRAM, ~78 GB
usable disk). Pairs with `notebooks/quickstart.ipynb`.

This is not a production-quality run — it is a smoke test that the training
loop works end-to-end on a small slice of 1° OM4 data, so external contributors
can experiment without HPC access.

To scale up, see `configs/samudra_om4_v2/`.
